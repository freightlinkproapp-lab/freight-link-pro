# Freight Link Pro

Freight Link Pro is a freight marketplace connecting brokers and drivers.

## Core rules
- Brokers post loads for free
- Drivers pay a 3% platform fee
- Built-in contracts only
- No DocuSign
- Broker can edit contracts before signing
- Contract locks after both signatures

## Structure
- `backend/` Node/Express API
- `ios_app/` iOS SwiftUI app
- `android_app/` Android app
- `database/` PostgreSQL/Supabase schema
- `website/` public website/privacy pages
- `docs/` setup guides
